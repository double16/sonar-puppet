To make it easier to add or reorder new attributes, parameters, etc., a trailing comma should be added after the last:

  • resource attribute
  • parameter definition
  • hash pair
  • selector case

file { 'foo':
  ensure => link,
  target => '/tmp/foo.bar'  # Noncompliant
}

class my_class (
  $param1,
  $param2                   # Noncompliant
) {  }

Compliant Solution

file { 'foo':
  ensure => link,
  target => '/tmp/foo.bar',
}

class my_class (
  $param1,
  $param2,
) { }

Exception

If a resource expression includes more than one resource body, the trailing comma is not mandatory.

file {
    'foo':
      ensure => link,
      target => '/tmp/foo',; # Compliant

    'bar':
      ensure => link,
      target => '/tmp/bar';  # Compliant
}
